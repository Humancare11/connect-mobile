## 🔐 SECURITY IMPLEMENTATION

### Tokens
- Stored in: Flutter Secure Storage (encrypted)
- iOS: Keychain
- Android: Keystore/EncryptedSharedPreferences

### User Data
- Stored in: SharedPreferences (local, non-sensitive)
- Contains: Name, email, role, mobile, DOB, gender, country

### Passwords
- Never stored locally
- Always hashed by server
- Strong requirements enforced client-side

### Session Management
- Atomic save/clear operations
- Automatic token attachment to requests
- Mounted checks before navigation
- Proper resource cleanup
